# Vicuña Server

The retained server surface is a host-side OpenAI-compatible HTTP service that:

- serves DeepSeek in `standard` mode
- relays to RunPod in `experimental` mode
- keeps Telegram/tool ownership on the host
- captures RL training data for experimental turns

For developer notes, see [README-dev.md](/Users/tyleraraujo/vicuna/tools/server/README-dev.md).
